# 
اكي اختبار CIA

منصة اختبار إلكترونية احترافية للتدقيق الداخلي وإدارة المخاطر والحوكمة والامتثال (GRC).

---

## نبذة عن المشروع

هذا المشروع عبارة عن نظام اختبارات تفاعلي يهدف إلى تدريب المتدربين في مجالات:
- التدقيق الداخلي
- إدارة المخاطر
- الحوكمة
- الامتثال
- التحضير لشهادة CIA

---

## مميزات النظام

- أسئلة عشوائية
- بنك أسئلة
- عرض النتيجة مباشرة
- إصدار شهادة نجاح
- طباعة الشهادة
- التحقق من الشهادات عبر QR Code

---

## ملفات المشروع

- index.html → الصفحة الرئيسية
- certificate.html → صفحة الشهادة
- verify.html → صفحة التحقق من الشهادة
- cia-question-bank.json → بنك الأسئلة
- certificate.png → صورة الشهادة
- admin-dashboard.html → لوحة التحكم
